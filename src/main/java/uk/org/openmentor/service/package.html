A web friendly model.

This package aims to represent the model in a more integrated and helpful way. Thus the model itself has no reference to the fact that it is persistent; hence the availability of eg a saveAssigment(Assignment ass) method in this package. This package also incorporates what (in the language of Torque) are Peer methods; those like getAssigments(String courseId) which are concerned with all assignments of a particular sort, rather than one particular instance.

The package also aims to be web-friendly. Typically this means having String arguments, simply because that is what is needed at the web level, alowing eg necessary conversions of an id to an int to be hidden here.
